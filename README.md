ZOMATO RESTAURANT ANALYSIS POWER BI PROJECT


PROJECT OVERVIEW:-
This project presents an interactive Power BI dashboard created using Zomato’s raw dataset. The dashboard provides meaningful insights into restaurant distribution, ratings, pricing, cuisines, and customer preferences across different locations. It is designed to help stakeholders understand food trends and make data-driven decisions.


FILES INCLUDED :-

zomato data file.xlsx – Raw dataset used for analysis and "ZOMATO DATA ANALYSIS DASHBOARD.pbit" – Power BI template file containing the complete dashboard





DASHBOARD FEATURES:-

*Interactive slicers for city, country, and cuisine

*Dynamic charts and KPIs

*Clean and user-friendly layout

*Drill-down analysis for better insights




PURPOSE OF THE PROJECT :-

This project was created to demonstrate-
*Data analysis skills

*Dashboard designing ability

*Business insight generation using real-world data




KEY INSIGHTS :-

*Restaurant Distribution Analysis:
Identifies the number of restaurants across different countries and cities, highlighting regions with the highest food service concentration.

Ratings Performance:
Analyzes average restaurant ratings to understand customer satisfaction levels and identify top-performing and low-performing locations.

*Price Range vs Ratings Relationship:
Shows how restaurant pricing impacts customer ratings, helping determine whether higher-priced restaurants necessarily receive better reviews.

*Cuisine Popularity:
Highlights the most popular cuisines based on restaurant count and customer engagement, helping understand food trends and preferences.

*Customer Engagement (Votes):
Examines voting patterns to measure customer interaction and popularity of restaurants across different regions.

*Online Delivery & Table Booking Trends:
Provides insights into how many restaurants offer online delivery and table booking services and how these services affect ratings and customer choice.

*Market Comparison:
Compares restaurant performance across different countries and cities to identify growth opportunities and competitive markets.
